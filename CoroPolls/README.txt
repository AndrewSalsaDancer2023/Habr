Сборка и запуск проекта. Предполагается что на ПК уже установлен CMake не ниже версии 3.10.
1. Находясь в корневой директории, ввести команду mkdir build && cd build
2. Проверить что директория build успешно создана и осуществлен переход в нее.
3. Ввести команду cmake .. для релизной сборки или команду cmake -DCMAKE_BUILD_TYPE=Debug .. для отладочной сборки
4. Завершить построение проекта командой cmake --build .
