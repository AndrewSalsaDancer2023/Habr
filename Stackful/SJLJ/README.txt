Сборка и запуск проекта. Предполагается что на ПК уже установлен CMake не ниже версии 3.10.
1. Находясь в корневой директории, ввести команду mkdir build && cd build
2. Проверить что директория build успешно создана и осуществлен переход в нее.
3. Ввести команду cmake .. для релизной сборки или команду cmake -DCMAKE_BUILD_TYPE=Debug .. для отладочной сборки
4. В СMakeLists.txt определиться со способом переключения контекста: 
	- при помощи - раскоментировать строку set(USE_SETJMP)
	- путем использования контекстных функций set(USE_CONTEXT ON)
	- с использованием ассемблерной реализации #set(USE_ASMCONTEXT ON)
Важно! В любом случае должна быть раскоментирована только одна из указанных строк.
5. Добавить макрос USE_SETJMP, USE_CONTEXT или USE_ASMCONTEXT в файл config.h.in.
Если в системе установлен Valgrind, то можно раскоментировать set(CORO_USE_VALGRIND ON) в СMakeLists.txt и добавить 
CORO_USE_VALGRIND в файл config.h.in.
6. Завершить построение проекта командой cmake --build .
